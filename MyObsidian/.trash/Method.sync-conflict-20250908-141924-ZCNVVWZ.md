### strip()
`strip()` removes spaces symbols from beginning and end

``` python
exapmle = '    Hello World!  '
print(example.strip())
```

### lower()
`lower()` converts all symbols from a string in lower case.

``` python
example = 'HELLO WORLD!'
print(example.lower())
```

### dict()
`dict()` is method for dictionary. It allow work with dictionary and make everything with it.

``` python
my_dict = {
	'name': 'John',
	'age': 30,
	'city': 'New York'
}
print(my_dict)

# get()
print(my_dict.get('age'))
print(my_dict.get('country'))
print(my_dict.get('country', 'Unknown'))

```
